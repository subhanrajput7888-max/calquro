How to Use the Student Calculator

Basic Operations

  • +, -, ×, ÷ - Basic arithmetic operations
  • % - Modulo (remainder) operation
  • = - Calculate result
  • AC - Clear all
  • CE - Clear entry
  • ⌫ - Delete last digit

Scientific Functions

  • sin, cos, tan - Trigonometric functions (degrees)
  • log - Base 10 logarithm
  • ln - Natural logarithm
  • √ - Square root
  • x², x³ - Square and cube
  • x^y - Power operation
  • x! - Factorial
  • 1/x - Reciprocal

Memory Functions

  • MS - Store value in memory
  • MR - Recall value from memory
  • M+ - Add to memory
  • M- - Subtract from memory
  • MC - Clear memory

Constants

  • π - Pi (3.14159...)
  • e - Euler's number (2.71828...)
  • ± - Change sign

💡 Tips for Students

  • Use keyboard shortcuts: numbers, +, -, *, /, Enter (=), Escape (AC)
  • For angle calculations, remember that trigonometric functions use degrees
  • Use parentheses in your mind for order of operations
  • The memory functions are great for storing intermediate results
  • Double-check your inputs before calculating complex expressions
